# Tag 1 – Konsolen-Ticketing-System mit Git
## Ziel
An diesem Tag wurde ein einfaches Konsolen-Ticketing-System in C# erstellt und mit Git versioniert.
## Aktionen
1. Neues C# Konsolenprojekt erstellt.
2. Klassenstruktur implementiert (`Ticket`, `Database`, `Program`).
3. CRUD-Methoden implementiert (Add, List, Update, Delete).
4. Git Repository initialisiert und erste Commits gemacht.
5. Verbindung zum Remote Repository hergestellt und Push erfolgreich durchgeführt.
## Notizen


- Alle Tickets werden aktuell nur in einer Liste gespeichert.
- Die ID-Vergabe erfolgt automatisch durch die `Ticket`-Klasse.
- Regelmäßig committen, um Änderungen nachzuvollziehen.
