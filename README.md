# Ticketing System – C# Konsolenanwendung
## Projektbeschreibung
Im Rahmen des AE-Grundlagen-Abschlussprojekts wurde ein Ticketing-System als C#-Konsolenanwendung entwickelt.
Das Projekt umfasst Planung, Umsetzung, Versionierung mit GitHub sowie Dokumentation.
## Ziel des Projekts
- Objektorientierte Programmierung (Klassen, Methoden, Konstruktoren)
- Saubere Projektstruktur und Codequalität
- Arbeiten mit GitHub
- Grundlegende Persistenz (JSON-Dateien)
- Agile Vorgehensweise (Daily Stand-ups, Backlog)
## Umfang & Abgrenzung
- Konsolenanwendung ohne GUI
- Rollen: Admin / User
- Tickets erstellen, anzeigen, speichern, löschen
## Technologien
- Sprache: C#
- Plattform: .NET
- Versionsverwaltung: GitHub
