# Backlog – Ticketing System
## User Stories
1. Als Benutzer möchte ich ein Ticket erstellen, um ein Problem zu erfassen.
2. Als Benutzer möchte ich alle meine Tickets anzeigen, um den Überblick zu behalten.
3. Als Admin möchte ich alle Tickets sehen, um den Status zu prüfen.
4. Als Benutzer/Admin möchte ich Tickets speichern und laden, um Daten zu sichern.
5. Als Admin möchte ich Tickets löschen können, um abgeschlossene Tickets zu entfernen.
6. Als Benutzer/Admin möchte ich Rollen unterscheiden, um unterschiedliche Berechtigungen zu haben.
